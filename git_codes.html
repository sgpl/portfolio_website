Common Git Commands

Commands + Descriptions:

Command Description
git init Initialize a git repo in the current directory.
git status Show status of the project.
git log Show a log of commits made.
git log --oneline Show a shorter summary of logs.
git commit -m "commit message" Commits added changes with the commit messages.
