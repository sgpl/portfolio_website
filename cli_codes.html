Common Commandline Commands

Common Commands:

Command Description
pwd Prints the present working directory.
cd ____ Changed the directory.
rm -rf ____ Removes a directory and all the files in it.
ls -la ____ Lists all the contents of a directory in long format.
mkdir ____ Makes a new directory.
